# Bemen

BEMEN Booking & Management

- Reservas online para clientes
- Login para clientes y administradores
- Gestión de reservas

Este proyecto está enfocado únicamente en la gestión de reservas, login de clientes y administradores, y la administración de citas. Todas las secciones de servicios han sido eliminadas para centrar la experiencia en la reserva y gestión.

Desarrollado para el bienestar masculino premium en Madrid.
